# プロトコル: 初回オンボーディング

## 発動条件

`config/profile.md` が存在しない、または「名前」が（未記入）のとき。他のどの依頼よりも優先する。

## 手順

0. `config/profile.md`・`config/tools.md` が存在しない場合（別PCでのclone直後など）は、まず両ファイルをテンプレート状態（全項目「（未記入）」、tools.md はMCP4ツールの雛形）で作成する
1. 「AI秘書のセットアップを始めます」と宣言し、以下を1〜2問ずつテンポよく質問する
   - お名前（呼び方）
   - 役職・立場
   - 主な責務（2〜3つ）
   - 報告の好み: 簡潔 / 詳細 / 要点+根拠
   - メール返信の文体: フォーマル / ややカジュアル / 相手による
2. 回答を `config/profile.md` に記入する
3. **Notion タスクDBのセットアップ**（フェーズ2）:
   - 「AI秘書タスク」DBを作成してよいか確認する（プロパティ仕様は `AI秘書セットアップ計画.md` の「Notion タスク DB」参照）
   - 承認後、Notion MCP（notion-create-database）でDBを作成し、DB URL/IDを `config/tools.md` に記入する
   - MCPでのDB作成に失敗した場合は、Notion UIでの手動作成（プロパティ仕様は計画書参照）を案内し、作成後にDB URL/IDを教えてもらって `config/tools.md` に記入する
   - ビュー（今日・未完了一覧・完了）はNotion側での手動作成を案内する
4. `config/tools.md` の残り項目を確認する
   - Gmail: 主に使うアドレス、下書きの署名
   - カレンダー: 対象カレンダー
5. 「セットアップが完了しました。何から始めますか？」で締める

## 注意

- 一度に全部質問しない
- 後から「プロフィールを更新して」で変更可能なことを伝える
