# プロトコル: 会議準備・議事録

## 発動条件

- 会議前:「〇〇の会議の準備して」「会議準備」
- 会議後:「議事録を作って」+ メモや書き起こしテキスト

## 会議前: 準備メモ

1. カレンダー（MCP）から会議情報（日時・参加者・議題）を取得
2. `areas/meeting-notes/` から同じ相手・案件の過去議事録を検索し、持ち越し事項（未完了のアクションアイテム）を抽出
3. 関連する `projects/` や `memory/people/` の情報を反映
4. 下記フォーマットで出力し、`areas/meeting-notes/YYYY-MM-DD_会議名_準備.md` に保存

```markdown
## 会議名（YYYY-MM-DD HH:MM）
### 参加者
### 議題
### 前回からの持ち越し
### 準備事項
```

## 会議後: 議事録

1. 渡されたメモ・書き起こしを構造化する
2. `areas/meeting-notes/YYYY-MM-DD_会議名.md` に保存
3. 自分（理さん）のアクションアイテムは **Notion タスクDBへの登録を提案**する（ソース: `会議`、承認後に登録）
4. 相手の新情報があれば `memory/people/` を更新

```markdown
## 会議名 議事録（YYYY-MM-DD）
### 決定事項
### アクションアイテム
| 担当 | タスク | 期限 |
|------|--------|------|
### 次回予定
### メモ
```

## 注意

- 決定事項とアクションアイテムを最優先で正確に。雑談・経緯は「メモ」に圧縮
- 期限が曖昧なアクションアイテムは【期限未定】と明示する
